Demonic Consumption Single Target Priority Rotation

If you have at least 1 Baleful Invocation and at least 1 Explosive Potential use the following opening sequence:
  1. Pre-cast Demonbolt
  2. Cast Doom, if talented
  3. Cast Bilescourge Bombers, if talented
  4. Cast Hand of Gul'dan
  5. Soul Strike, if talented
  6. Build to 5 Soul Shards
  7. Implosion
  8. Cast Grimoire: Felguard if talented
  9. Build to 5 Soul Shards
  10. Cast Summon Vilefiend, if talented
  11. Build to 5 Soul Shards
  12. Cast Call Dreadstalkers
  13. Build to 5 Soul Shards
  14. Cast Hand of Gul'dan
  15. Build to 3 Soul Shards
  16. Cast Hand of Gul'dan
  17. Cast Demonic Strength or Doom (again), if talented, or Shadowbolt
  18. Cast Summon Demonic Tyrant
Regular priority:
  1. Maintain Explosive Potential by casting Implosion with at least 3 Wild Imps out
  2. Cast Summon Demonic Tyrant on cooldown with at least 2 sets of Wild Imps out
  3. Cast Demonic Strength or Bilescourge Bombers on cooldown, if talented
  4. Cast Grimoire: Felguard if talented, on cooldown
  5. Cast Summon Vilefiend on cooldown, if talented
  6. Cast Call Dreadstalkers on cooldown
  7. Cast 3 Soul Shards Hand of Gul'dan whenever possible
  8. Cast Demonbolt or Shadow Bolt to generate Soul Shards
Tyrant Setup:

For Tyrants after opener you will want to have 2 or more sets of imps out to maximize Demonic Consumption. This is best achieved by going into each Tyrant with 2 or more cores and a fresh Explosive Potential buff. Do not delay Tyrant for Grimoire Felguard or vice versa unless there is only 1 cast of each remaining in an encounter.

  1. Build to 5 Soul Shards
  2. Implosion if Explosive Potential will run out before Tyrant cast
  3. Grimoire: Felguard if talented and off cooldown
  4. Build to 5 soul shards with shadowbolt or Soul Strike if talented
  5. Summon Vilefiend if talented and off cooldown
  6. Build to 5 Soul Shards with shadowbolt or Soul Strike
  7. Call Dreadstalkers
  8. Build to 5 Soul Shards with Shadowbolt, Soul Strike, or Demonbolt if you didn't have a Demonic Calling proc and have 3 or more Demonic Cores
  9. Hand of Gul'Dan
  10. Demonbolt
  11. Hand of Gul'Dan
  12. Demonbolt
  13. Hand of Gul'Dan
  14. Hand of Gul'Dan again to spend any remaining shards due to Conduit or Demonic Meteor procs
  15. Wait ~0.5s if you did not get procs for an extra HoG
  16. Summon Demonic Tyrant

Sacrificed Souls Single Target Priority Rotation

Opening priority:
  1. Pre-cast Demonbolt
  2. Cast Doom, if talented
  3. Cast Demonic Strength or Bilescourge Bombers, if talented
  4. Cast Grimoire: Felguard if talented
  5. Cast Summon Vilefiend, if talented
  6. Cast Shadowbolt - not necessary if talented into only 1 of either Grimoire: Felguard or Summon Vilefiend
  7. Cast Call Dreadstalkers
  8. Cast Shadowbolt
  9. Cast Shadowbolt
  10. Cast Shadowbolt - not necessary if talented into only 1 of either Grimoire: Felguard or Summon Vilefiend
  11. Cast Hand of Gul'dan
  12. Cast Summon Demonic Tyrant
Regular priority:
  1. Cast Summon Demonic Tyrant on cooldown
  2. Cast Demonic Strength or Bilescourge Bombers on cooldown, if talented
  3. Cast Grimoire: Felguard if talented, on cooldown
  4. Cast Summon Vilefiend on cooldown, if talented
  5. Cast Call Dreadstalkers on cooldown
  6. Cast 3 Soul Shards Hand of Gul'dan whenever possible
  7. Cast Demonbolt or Shadow Bolt to generate Soul Shards

Two+ Target Priority Rotation

  1. Pre-cast Demonbolt
  2. Cast Demonic Strength or Bilescourge Bombers, if talented
  3. Cast Grimoire: Felguard if talented, on cooldown
  4. Cast Summon Vilefiend if talented, on cooldown
  5. Cast Call Dreadstalkers, on cooldown
  6. Cast Summon Demonic Tyrant, on cooldown with as many of the above cooldown demons and Wild Imps out
  7. Cast Doom on all possible targets, if talented and mobs will live for at least 30s
  8. Build to 3 Soul Shards
  9. Cast Hand of Gul'dan
  10. If targets are stacked already enter Implosion Rotation
    With 2 or more stacks of Demonic Core:
    • Build to 5 Soul Shards
    • Cast Hand of Gul'dan
    • Build 4 Soul Shards using Demonbolt
    • Cast Hand of Gul'dan
    • Build to 3 Soul Shards using Demonbolt
    • Cast Hand of Gul'dan
    • Cast Shadow Bolt
    • Cast Shadow Bolt
    • Cast Implosion
    Without a stack of Demonic Core:
    • Build to 5 Soul Shards
    • Cast Hand of Gul'dan
    • Build to 3 Soul Shards
    • Cast Hand of Gul'dan
    • Cast Shadow Bolt or Demonbolt if you have a stack of Demonic Core
    • Cast Shadow Bolt
    • Cast Implosion
  11. Fill with Demonbolt if you have available stacks of Demonic Core - alternately fill with Shadow Bolt
